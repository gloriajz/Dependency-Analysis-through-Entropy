This project introduces GNDD (Generalized Nonlinear Dependency Distance), an entropy-based distance metric developed to detect linear and nonlinear dependencies in high-dimensional and noisy datasets. GNDD addresses limitations of mutual information and correlation, offering robustness, interpretability, and scalability.

  • Design a normalized, metric-compliant dependency distance using entropy
  • Capture nonlinear and multivariate relationships across mixed-type variables
  • Enhance robustness under noise and small sample sizes
  • Apply the GNDD metric to real financial data and synthetic simulations

ðŸ§® Methodology

  • GNDD Derivation: Built from normalized mutual information and entropy, satisfying metric properties
  • GNDDrobust: Includes noise thresholding to filter spurious dependencies
  • Dynamic GNDD: Uses sliding windows to track time-varying relationships
  • Multivariate GNDD: Extends to multiple variables using joint entropy estimation
